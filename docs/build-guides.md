## Property Style Guide

| Group | Material Theme | Style Direction |
|---|---|---|
| A | Oak Planks | Early-game village lane, simple and cozy. |
| B | Packed Ice | Clean light-blue roads, river and frost accents. |
| C | Cherry Planks | Busy town center look with warmer color contrast. |
| D | Cut Copper | Badlands/mesa vibe, weathered orange textures. |
| E | Redstone Block | Nether-tech feel with blackstone and red lighting. |
| F | Gold Block | Bright savanna district with warm acacia tones. |
| G | Emerald Block | Jungle-trader district with bamboo and lush details. |
| H | Diamond Block | Endgame district with polished, premium finishes. |

## Special Tile Build Notes

| Tile Type | Build Notes |
|---|---|
| `Spawn` | Make this the cleanest hub tile; clear sightlines and score/economy displays. |
| `Dungeon / Sent to Dungeon` | Use bars, pistons, and controlled entry/exit lanes for status clarity. |
| `Adventure Cards` | Include a card draw mechanism and visible resolution indicator. |
| `Minecart Depot` | Keep depot silhouette consistent on all four sides for fast recognition. |
| `Utilities` | Use functional-looking machinery and redstone lamps to imply system control. |
| `Trade Tax / Beacon Tax` | Build obvious payment pedestals with hopper input points. |
| `Campfire Rest` | Neutral rest tile with no interactive economic mechanism. |
