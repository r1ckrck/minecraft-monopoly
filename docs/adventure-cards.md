# Adventure Cards

Back to index: [`README.md`](../README.md)

## Draw Rule

- Adventure Cards must be used immediately when drawn.
- Exception: keep-style cards can be stored for later use (currently `Get Out of Dungeon Free`).

## Deck Spec

| Card Name | Effect | Duplicates |
|---|---|---|
| `Village Gift` | Receive `6 E` from the bank. | 3 |
| `Emerald Vein` | Receive `1 EB` from the bank. | 1 |
| `Spawn Festival` | Move to `Spawn`; collect Spawn reward. | 2 |
| `Minecart Transfer` | Move to nearest `Minecart Depot`; if owned, pay tribute. | 2 |
| `Utility Callout` | Move to nearest utility; if owned, pay tribute. | 2 |
| `Speed Potion` | Move forward `3` spaces. | 2 |
| `Wrong Portal` | Move backward `2` spaces. | 2 |
| `Dungeon Patrol` | Go directly to `Sent to Dungeon` (no Spawn reward). | 2 |
| `Get Out of Dungeon Free` | Keep this card; use once to leave Dungeon without fine. | 1 |
| `Beacon Service Fee` | Pay `15 E` to the bank. | 1 |
| `Tool Repairs` | Pay `6 E` to the bank. | 1 |
| `Market Day` | Collect `2 E` from each other player. | 1 |
| `Ender Pull` | Choose `1` player; move them to your current space. | 1 |
| `Spleefed` | Choose `1` player; move them backward `3` spaces. | 2 |
| `Dungeon Warrant` | Choose `1` player; send them to `Sent to Dungeon` (no Spawn reward). | 1 |
| `Bandit Raid` | Choose `1` player; they pay you `8 E` (or all they can). | 2 |

Deck size: `26` cards total.
