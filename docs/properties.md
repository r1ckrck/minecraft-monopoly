## Set and Upgrade Rules

- Full set bonus (own both in group): `2x` base tribute.
- All properties follow the same upgrade path: `2 Outposts -> 1 Stronghold`.
- Build cost per upgrade step: `(Property CT cost)` on that tile.
- Tribute multipliers: `No set/no build = 1x`, `Set only = 2x`, `1 Outpost = 3x`, `2 Outposts = 4x`, `Stronghold = 6x`.

## Material Progression (A to H)

`Oak Planks -> Packed Ice -> Cherry Planks -> Cut Copper -> Redstone Block -> Gold Block -> Emerald Block -> Diamond Block`
