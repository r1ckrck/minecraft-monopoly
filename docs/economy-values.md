## Currency + Exchange

| Economy Item | Value |
|---|---|
| Emerald conversion | `1 EB = 9 E` |
| Buy `1 CT` | `1 EB` |
| Cash out `1 CT` | `8 E` (1 Emerald fee) |
| Starting funds per player | `6 EB + 18 E` (total `72 E` value) |
| Passing `Spawn` reward | `1 EB + 3 E` (`12 E`) |
| `Trade Tax` | `10 E` |
| `Beacon Tax` | `15 E` |
| Dungeon release fine | `1 EB` (`9 E`) |

## Tribute Rules

| Property Group | Base Tribute |
|---|---|
| A | `2 E` |
| B | `4 E` |
| C | `6 E` |
| D | `8 E` |
| E | `10 E` |
| F | `12 E` |
| G | `14 E` |
| H | `16 E` |

- Full color set (both properties in group): `2x` base tribute.
- Tribute multipliers on a property: `No set/no build = 1x`, `Set multiplier = 2x`, `1 Outpost = 3x`, `2 Outposts = 4x`, `Stronghold = 6x`.
- Outpost or Stronghold cost per build: `(Property CT cost)` on that tile.
- Minecart Depot tribute by count owned: `1 = 6 E`, `2 = 12 E`, `3 = 20 E`, `4 = 30 E`.
- Utility tribute: `1 utility = 8 E`, `2 utilities = 18 E`.
