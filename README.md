# Minecraft Monopoly

A fully playable Minecraft-themed Monopoly variant built for redstone-friendly gameplay.

## Game Snapshot

- Board size: `32` spaces (`28` regular + `4` corners).
- Money: `Emeralds (E)` and `Emerald Blocks (EB)`.
- Purchase currency: `Claim Tokens (CT)` for properties, utilities, and minecart depots.
- Property upgrade path (all groups): `2 Outposts -> 1 Stronghold`.
- Adventure Cards: resolve immediately when drawn (except keep cards).
- Win condition: last non-bankrupt player.

## Setup (New Players)

1. Build the board from [`docs/board-layout.md`](docs/board-layout.md).
2. Prepare bank and starting funds from [`docs/economy-values.md`](docs/economy-values.md).
3. Shuffle and place the Adventure deck from [`docs/adventure-cards.md`](docs/adventure-cards.md).
4. Use [`docs/rules-reference.md`](docs/rules-reference.md) for turn flow and edge cases.
